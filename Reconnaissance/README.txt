🔎 Whois Lookup Tool

A simple yet powerful Whois Lookup Tool written in Python.
It fetches details about a domain such as:

Registrar

Creation Date

Expiry Date

Name Servers

🎨 Includes cool ASCII Art Banner and colored output.
✨ Created with ❤️ by Syed Ashir.

📦 Requirements

Install dependencies using:

pip install -r requirements.txt


requirements.txt includes:

whois → Domain info fetch karne ke liye

colorama → Colored output

pyfiglet → ASCII Art

▶️ Usage

Clone this repository:

git clone https://github.com/your-username/RedTeam-Toolkit.git


Navigate into the tool directory:

cd RedTeam-Toolkit/Whois-Lookup-Tool


Run the script:

python whois_lookup.py


Enter the domain when asked, for example:

Enter domain: google.com


Output Example:

========================================
  Whois Lookup Tool  
  Created by Syed Ashir  
========================================

Domain: google.com  
Registrar: MarkMonitor Inc.  
Creation Date: 1997-09-15  
Expiry Date: 2028-09-14  
Name Servers: ns1.google.com, ns2.google.com
